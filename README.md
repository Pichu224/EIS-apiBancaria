# API Bancaria

Proyecto de ejemplo para una API bancaria con Spring Boot.

## Descripción

Esta aplicación expone endpoints básicos de usuarios, cajas, movimientos y transferencias.

## Requisitos

- Java 21
- Gradle Wrapper (`gradlew` / `gradlew.bat`)
- PostgreSQL para ejecución real (solo si se usa la aplicación completa)

## Ejecución

Desde la raíz del proyecto:

```powershell
.\gradlew.bat bootRun
```

## Pruebas

Ejecutar la suite completa de pruebas unitarias:

```powershell
.\gradlew.bat test
```
